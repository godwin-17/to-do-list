## Features

- <b>Adding</b> a Todo

- <b>Marking</b> a Todo as done or not

- <b>Deleting</b> Todo

- Todos are saved in <b>local storage</b>

- <b>Marked todos</b> are saved at the bottom and have a <s>strikethrough</s> on them.

## Component Heirarchy

- Todo App
  - Todo Modal
  - Todo List
